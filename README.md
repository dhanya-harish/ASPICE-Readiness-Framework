# ASPICE Readiness Capstone — Starter Dataset

## Important
The CSV files in this repository are **synthetic, illustrative data** created only to establish the capstone project structure, demonstrate reproducibility, and support early notebook screenshots. They are **not measured ASPICE assessment results** and must not be presented as empirical findings.

## Files
- `data/raw/aspice_sprint_metrics_raw_sample.csv` — 120 sprint-level sample observations with a small number of missing values.
- `data/processed/aspice_sprint_metrics_processed_sample.csv` — cleaned version for initial EDA and pipeline testing.
- `data/data_dictionary.csv` — variable definitions.

## Recommended use at synopsis stage
1. Load the raw sample in a notebook.
2. Display `head()`, `shape`, `info()`, descriptive statistics, and missing-value counts.
3. Document that the production study will replace or augment this synthetic sample with eligible public repository data.
4. Keep all extraction and preprocessing scripts reproducible.

## Ethical and academic note
The readiness score and compliance indicators are generated using transparent illustrative rules. They do not represent official VDA QMC assessment outcomes and do not replace qualified ASPICE assessor judgment.
